## 升级指南

### 从v0.3.8升级到v0.3.9+
1. 停止旧版本：`./process-tracker-old stop`
2. 替换可执行文件
3. 启动新版本：`./process-tracker start --web`
4. 数据自动兼容，无需迁移

### 数据格式兼容性
- v7 (18字段)：支持CPU归一化和内存百分比
- v6 (17字段)：支持内存百分比
- v5 (16字段)：基础格式

所有版本数据可混合使用，自动识别格式。
